# Kashio Figma Landing Page

Pixel-accurate frontend recreation of the supplied Kashio landing-page screenshots.

## Stack

- Next.js 16 App Router
- JavaScript
- Tailwind CSS 4
- Shadcn-style reusable UI components
- Lucide icons
- Next.js Image optimization

The project is frontend-only. It contains no API routes, authentication, database, or server-side business logic.

## Run locally

```bash
npm install
npm run dev
```

Open `http://localhost:3000`.

## Production build

```bash
npm run build
npm start
```

  ui/
